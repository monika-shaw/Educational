If you are Firefox user , then please note that Firefox is not supported backdrop-filter by default. For enable this option follow this below steps :
1. Open new Firefox tab then type about:config in search bar and click "accept the risk and continue".
2. Enter “layout.css.backdrop-filter.enabled” in search preference name and you can see value is false in this preference , so you need to change it to true. Next, enter "gfx.webrender.all" and set it to true similarly and check codepen now.
